## How to use
 1. Install XAMPP.  
 2. Copy the repository in the folder htdocs of XAMPP.  
 3. Run complain_db.sql which is in [here](/HTML/database).
 4. Make changes in [here](/HTML/library/config.php) according to your database.
 5. Run the localhost.
 6. __Voila__ .
